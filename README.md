# AI Backlog Tool — Prototype

Proof-of-concept for an AI-powered backlog tool that connects customer feedback sentiment to value-estimated work items. All data is mock. All AI is simulated.

## What it demonstrates

Three product pillars that no existing tool combines:

1. **Sentiment-to-Backlog Loop** — Ingest feedback from support, sales, and NPS. Surface trends. Auto-suggest prioritized work items with dollar-value estimates.
2. **Cross-Team Scanning** — Detect duplicate/conflicting work across teams. Map dependencies. Prevent wasted effort.
3. **Codebase Enrichment** — Break user stories into developer tasks with file-level code context and QA edge cases.

## Quick start

```
npm install
npm run dev
```

Open http://localhost:5173

## Stack

React 18, TypeScript, Vite, Tailwind CSS, recharts, vitest. No backend.

## Status

This is a wireframe-fidelity prototype built to validate the concept. Not production code.
